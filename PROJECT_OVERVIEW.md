# EmergencyAlert - Project Overview

## What is this app?
A mobile emergency alert application built for a volunteer
emergency response team of 10-15 people.

## The Problem
Team members often have their phones on silent or sleep mode
during the night. When an emergency occurs, the team leader
has no reliable way to alert everyone instantly.

## The Solution
A mobile app where authorized users press a single button,
and ALL team members receive a loud alert that bypasses
silent mode and DND (Do Not Disturb).

## User Roles

### Super Admin (1 person)
- Full system access
- Can send alerts to all teams or specific team
- Can add/remove users
- Can assign users to teams
- Can promote users to Manager role
- Manages the entire system

### Manager (up to 2 people)
- Can send alerts to all teams or specific team
- Cannot manage users or system settings

### Member (regular team members)
- Receives alerts only
- No sending permissions

## Access Control
- App is closed system - users join via one-time invite code
- Invite codes are generated by Super Admin only
- No public registration

## Alert Levels

### 🟢 Green Alert
- Severity: Informational
- Meaning: Not an emergency. General update or app test.
- Behavior: Standard notification sound

### 🟠 Orange Alert
- Severity: Medium
- Meaning: Active event - anyone who can respond, please come
- Behavior: Loud notification, bypasses silent mode

### 🔴 Red Alert
- Severity: Critical
- Meaning: Real emergency - everyone must respond immediately
- Behavior: Maximum volume, bypasses silent mode AND DND,
  repeating alarm until acknowledged

## App Language
Hebrew (עברית) - all UI text must be in Hebrew

## Tech Stack
- Framework: Expo (React Native)
- Notifications: Firebase Cloud Messaging (FCM)
- Backend/DB: Firebase Firestore
- Auth: Firebase Anonymous Auth + Invite Code
- Target: Android (primary), iOS (secondary)

## Key Technical Challenge
The notification MUST override silent mode and DND
on both Android and iOS.
- Android: FCM high-priority channel, importance level MAX
- iOS: Requires Apple Critical Alerts entitlement
